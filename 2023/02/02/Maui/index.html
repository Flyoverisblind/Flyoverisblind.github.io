什么是Maui

.NET 多平台应用 UI (.NET MAUI) 是一个跨平台框架，用于使用 C# 和 XAML 创建本机移动和桌面应用。
使用 .NET MAUI，可从单个共享代码库开发可在 Android、iOS、macOS 和 Windows 上运行的应用。

布局控件

Grid

Grid是控制行和列的控件,在maui中有了更方便设置行和列的方法

原来在wpf中设置两行三列

而现在在Maui中设置两行三列使用RowDefinitions和ColumnDefinitions就可以，引号里有几个值代表分成几列或几行
